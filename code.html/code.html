    Práctica 2: ISO-Prolog



    Introducción

    Este módulo implementa un compresor y descompresor de secuencias de caracteres. El objetivo es reducir la longitud de una secuencia representando patrones repetitivos de forma más concisa. La compresión es con pérdidas si la secuencia comprimida es más larga que la original, pero el objetivo es encontrar la representación más corta.

    Representación de datos

    Las secuencias son listas de Prolog. Los elementos de estas listas sin comprimir estan compuestas por caracteres.

    Una secuencia comprimida también es una lista, pero puede contener:

    • Átomos originales.
    • Pares [Elemento, Num], donde Elemento es un átomo y Num es el número de veces que se repite.
    • Grupos complejos ['<', S1, S2, ..., Sk, '>', Num], donde S1...Sk es una subsecuencia (que puede estar ya comprimida o ser una mezcla de átomos y grupos simples) y Num indica cuántas veces se repite toda la subsecuencia S1...Sk.
    La descompresión revierte estos formatos a la secuencia original de átomos.

    Funcionalidades

    El módulo implementa las siguientes funcionalidades principales:

    1. Compresión de secuencias: Busca la representación más corta de una secuencia dada, utilizando memoización para optimizar el proceso.
    2. Descompresión de secuencias: Reconstruye la secuencia original a partir de su forma comprimida.
    3. Predicados auxiliares para dividir secuencias, agrupar elementos repetidos, y encontrar la compresión óptima entre varias candidatas.

    Ejemplos de uso

    A continuación, se presenta un ejemplo de cada funcionalidad principal del módulo.

    split/3 - Dividir una secuencia: 

    ?- split([a,b,c], Parte1, Parte2).
Parte1 = [a],
Parte2 = [b, c] ? ;
Parte1 = [a, b],
Parte2 = [c] ? ;

    group/3 - Formar un grupo comprimido: 

    ?- group([a,b,c,d,e], 3, Grupo).
Grupo = ['<', a, b, c, d, e, '>', 3]

    is_repeated/3 - Verificar si una secuencia es una repetición: 

    ?- is_repeated([a,b,a,b,a,b], [a,b], Num).
Num = 3

    simple_repetition/2 - Compresión por repetición simple: 

    ?- simple_repetition([x,y,x,y,x,y], Comprimida).
Comprimida = ['<', x, y, '>', 3]

    repetition/2 - Compresión por repetición (con compresión interna): 

    ?- repetition([a,a,a,a], Comprimida) 
Comprimida = [a, 4] ? ; 
Comprimida = ['<',a,a,'>',2]

    compress/2 - Compresión completa de una secuencia: 

    ?- compress([a,b,b,b,b,a,b,b,b,b], Comprimida).
Comprimida = ['<', a, b, 4, '>', 2]

    min_list/2 - Encontrar la lista más corta en una lista de listas: 

    ?- min_list([[a,b,c],[a,b],[a,b,a,b]], Min).
Min = [a, b]

    decompress/2 - Descompresión de una secuencia: 

    ?- decompress(['<',a,a,b,b,'>',3,c], Descomprimida).
Descomprimida = [a, a, b, b, a, a, b, b, a, a, b, b, c]

    expand_sequence/3 - Expandir una secuencia repetidamente: 

    ?- expand_sequence([a,b,c], 3, SecuenciaExpandida).
SecuenciaExpandida = [a, b, c, a, b, c, a, b, c]
